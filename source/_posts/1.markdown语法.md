## 1.2文字(文本)显示

1. 上下标：

​     `x^2^`

​    `H~2~O`

   H~2~O

2.文本居中

代码:

```

```

效果：
<center>内容</kbd>

代码：

```
<kbd>内容</kbd>
```

<kbd>内容</kbd>

```
<b>加粗</b>

```

# 2.列表

## 2.1无序列表

代码：

 `*/-/+ + 空格`

效果：

1.只有同一级别：

+ 1

+ 2

+ 3

	

	2.子集类

	- 1
		- 2
			- 3

	快捷键：ctrl+shift+]

## 2.2有序列表

代码：

`数字+.+空格`

效果：

1. 第一个标题

2. 第二个标题

3. 第三个标题

	- 子内容1
	- 子内容2

4. 第四个标题
	快捷键为ctrl+shift+[

	## 2.3任务列表

	代码：

	`- [ ]吃早饭`

	`- [x]背单词`
	效果：

	- [ ] 吃早饭

	- [ ] 背单词

快捷键：ctrl+shift+x

# 6.脚注

对文本进行解释说明
代码：

```
[^文本]
[^文本]：解释说明
```

效果：

​    c++[^①]

c[^②]

[^①]:这是一种编程语言
[^②]: 这是一种简单编程语言
